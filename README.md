# RedNote-WebEmbed · 小红书 App 原生按钮内嵌 H5 定制方案（Android & iOS 双端）

> **关键词 ** 小红书逆向、RedNote 定制、APK 反编译、IPA 重签名、Frida Gadget 注入、iOS Dylib Hook、Android WebView 内嵌、原生按钮跳转 H5、App 二次开发、Mobile Reverse Engineering、iOS Tweak、Android Smali 修改、无越狱注入、Sideload IPA、App 商业化定制


## ✨ 项目简介

本项目是一套面向主流社交内容 App 的 **原生入口 → 自定义 H5 页面** 的内嵌替换方案，已在 **Android APK** 与 **iOS IPA** 双端完整落地，并稳定通过线上环境验证。

通过对目标 App 的原生导航入口（如底部 Tab、顶部按钮、商城入口等）进行**无感劫持**，在不破坏 App 整体稳定性与用户体验的前提下，将点击行为跳转至我们自有的 H5 业务页面，并完成**用户身份透传**，从而实现：

- 🎯 **精准流量承接** —— 利用原生 App 流量入口驱动业务转化
- 🔐 **用户信息穿透** —— 自动携带登录态与身份参数，H5 端免二次登录
- 🧩 **页面任意替换** —— 后端远程下发目标 URL，无需重新打包
- 🛡️ **稳定抗更新** —— 已形成成熟迭代流程，可快速适配目标 App 的版本升级
