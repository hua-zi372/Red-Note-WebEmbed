# 📦 Samples · 公开代码片段

本目录仅存放**脱敏后的通用示例代码**，用于向外界展示本项目的工程化思路。

## ⚠️ 重要声明

- 以下代码均为**行业标准写法 / 教程级片段**，不涉及本项目的核心 Hook 点、真实类名、注入时机、绕过方案。
- 所有示例中的 URL、包名、类名、账号均为**占位符**。
- 真正可以产出 APK / IPA 的关键实现**不在此仓库**，请通过 Telegram 洽谈。

## 目录结构

```
samples/
├── android/
│   └── WebViewActivityDemo.java      # 标准 Android WebView 承载页
├── ios/
│   └── WebViewDemo.m                 # 标准 iOS WKWebView 承载页
├── server/

└── utils/
    └── UrlSafeBase64Demo.java        # URL-safe Base64 编解码工具
```
